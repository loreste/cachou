# Changelog

## 0.2.0

- Initial extension release aligned with CachouJS 0.2.0.
- `.cachou` language grammar (script / style / template / `{expressions}`).
- Snippets for SFCs and JavaScript/TypeScript consumers.
- Completions and hover for runtime APIs and directives.
- Compile current file & workspace component directories.
- Compile on save, status bar action, create-component scaffold.
- Compiler diagnostics with unsaved-buffer support.
- Commands to open workspace documentation (docs, API, how-tos, examples).
