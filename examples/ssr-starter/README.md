# SSR starter

Minimal Node server that uses `renderToStringAsync`, `dehydrate`, and `getSSRHead`.

```bash
# from repo root
node examples/ssr-starter/server.mjs
# open http://127.0.0.1:8787
```

See also [docs/DEPLOY.md](../../docs/DEPLOY.md) and [docs/how-to/ssr-and-hydration.md](../../docs/how-to/ssr-and-hydration.md).
